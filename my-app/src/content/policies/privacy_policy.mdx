#### Cookies:

<Divider />

A "cookie" is a text file containing a small amount of information which our server downloads to user's personal computer when user visit our website. The file contains a unique number so that our server knows which PC it is talking to. We use a session cookie to know whether user have logged in as an old student or not. Some cookies are allocated to user's PC only for the duration of user's visit to a website, and these are called session based cookies. These are automatically deleted when user close down his/her browser. <br/><br/>

Users can adjust the settings on their computer to decline any cookies if they wish. This can be done by activating the reject cookies setting on user’s computer.
Third party links in this website may also use cookies, over which VRI has no control. <br/><br/>
